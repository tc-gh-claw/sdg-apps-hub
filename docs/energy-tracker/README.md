# ⚡ 綠電追蹤 | Green Energy Tracker

## SDG 7: 經濟適用的清潔能源

一款簡潔優雅的能源追蹤應用，幫助用戶記錄家庭用電、了解電器耗電量，並鼓勵使用可再生能源。從個人層面推動清潔能源使用意識。

## 功能特性

### 🎯 核心功能

1. **即時用電儀表**
   - 視覺化半圓形儀表顯示今日用電
   - 動態填充效果，一目瞭然
   - 目標達成百分比

2. **快速電器記錄**
   - 常用電器一鍵記錄（冷氣、洗衣機、電腦等）
   - 自動計算耗電量（瓦數 × 時間）
   - 自訂電器與使用時間

3. **綠電比例追蹤**
   - 記錄太陽能等可再生能源使用
   - 綠電百分比徽章
   - 鼓勵清潔能源使用

4. **用電統計分析**
   - 本週用電趨勢圖表
   - 與上週比較
   - 預估電費金額

5. **節能建議**
   - 根據用電模式提供建議
   - 高峰時段提醒
   - 電器節能技巧

## 使用方式

直接在瀏覽器打開 `index.html` 即可使用。

### 快速開始
1. 設定每日用電目標（建議 8-15 kWh）
2. 每次使用電器後點擊對應按鈕記錄
3. 記錄綠電使用（如太陽能）
4. 查看儀表了解用電狀況
5. 追蹤節能進度
