## 關於此項目

每小時自動生成一款針對聯合國可持續發展目標 (SDGs) 的應用程式原型，涵蓋 17 個目標領域，從消除貧困到氣候行動。

**設計原則：**
- 每款應用都有明確的視覺設計參考
- 功能實用且可立即使用
- 純前端實現，易於部署
- 響應式設計，優先考慮移動端體驗
