# 🌱 低碳通勤追蹤器 | Low-Carbon Commute Tracker

## SDG 13: 氣候行動

一款簡約優雅的碳足跡追蹤工具，幫助用戶記錄日常通勤方式，了解不同交通工具的碳排放差異，並鼓勵選擇更環保的出行方式。

---

## 設計理念

**視覺風格參考：**
- **Dieter Rams** 的極簡主義設計哲學
- **日本傳統美學** 的侘寂與留白
- 應用如 **Forest**、**Headspace** 等正念應用的平靜視覺語言

**設計原則：**
- 少即是多 (Less but better)
- 大地色系：森林綠、米白、石灰色
- 大量留白，呼吸感
- 圓角、柔和陰影

---

## 功能特性

### 🎯 核心功能
1. **即時碳足跡計算** - 根據交通方式與距離自動計算 CO₂ 排放
2. **減碳成就系統** - 對比私家車基準，顯示每次行程節省的碳排放
3. **本週進度追蹤** - 視覺化環形進度條，目標 20 程低碳出行
4. **本地儲存** - 使用 LocalStorage 保存數據，無需帳號

### 🚇 支援交通方式
| 方式 | 圖示 | 碳排放 (g/km) |
|------|------|---------------|
| 步行 | 🚶 | 0 |
| 單車 | 🚲 | 0 |
| 港鐵 | 🚇 | 27 |
| 巴士 | 🚌 | 89 |
| 的士 | 🚕 | 192 |
| 私家車 | 🚗 | 171 |

*數據來源：香港環境保護署*
