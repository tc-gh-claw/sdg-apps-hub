# 💧 智水管家 | Water Wise

## SDG 6: 清潔飲水和衛生設施

一款優雅直觀的家庭用水追蹤應用，幫助用戶記錄日常用水、設定節水目標，並提供實用節水建議。從個人層面推動可持續水資源管理。

---

## 設計理念

**視覺風格參考：**
- **深澤直人 (Naoto Fukasawa)** - 無意識設計，直覺式體驗，彷彿物品本來就該在那裡
- **Otl Aicher** - 慕尼黑奧運會設計系統的清晰圖標與色彩邏輯
- **Sonos 應用** - 乾淨、無干擾的界面哲學

**設計原則：**
- 海洋漸層：深海藍到淺海綠的流動感
- 水波動畫：核心的視覺隱喻，呼吸般的動態
- 大量留白，如同水面的寧靜
- 溫暖但不過度熱情的色調

## 功能特性

### 🎯 核心功能

1. **即時用水追蹤**
   - 水波視覺化呈現今日總用水量
   - 六種常用活動一鍵記錄（淋浴、洗衣、洗碗等）
   - 自訂用水量輸入

2. **智能節水目標**
   - 每日目標：150 公升（香港人均約 220L）
   - 視覺化進度條
   - 目標達成百分比顯示

3. **節水成就統計**
   - 對比香港人均用水的節省比例
   - 估算節省水費金額
   - 每日用水目標達成率

4. **每日節水錦囊**
   - 輪播實用節水技巧
   - 基於當地水資源數據的建議
   - 從小習慣到大影響的提醒

5. **本地數據儲存**
   - 使用 LocalStorage 保存所有數據
   - 自動每日重置記錄
   - 無需帳號，開箱即用

## 使用方式

直接在瀏覽器打開 `index.html` 即可使用。

### 快速開始
1. 每次用水後點擊對應活動按鈕
2. 或使用自訂輸入記錄特殊用水
3. 查看水波視覺化了解今日用水量
4. 追蹤目標進度，實現節水目標
5. 閱讀每日錦囊，學習新技巧
