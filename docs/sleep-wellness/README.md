   - 快速記錄就寢與起床時間

## 使用方式

直接在瀏覽器打開 `index.html` 即可使用。

### 快速開始
1. 設置你的理想睡眠時間（建議 7-9 小時）
2. 每晚睡前點擊「開始睡眠」
3. 早晨起床後點擊「起床」並評分
4. 查看統計，了解你的睡眠模式
5. 閱讀每日錦囊，改善睡眠質量
