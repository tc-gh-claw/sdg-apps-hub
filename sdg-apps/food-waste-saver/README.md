# 🌿 惜食管家 | Food Waste Saver

## SDG 12: 負責任消費和生產

一款優雅直觀的食物管理應用，幫助用戶追蹤家中食物的保質期，減少浪費，並根據現有食材推薦食譜。從個人層面推動可持續消費。

## 功能特性

### 🎯 核心功能

1. **智能過期提醒**
   - 自動計算並顯示剩餘天數
   - 分級警示：安全(綠)、注意(黃)、緊急(紅)
   - 即將過期食物優先顯示

2. **快速新增**
   - 快捷輸入欄位
   - 常用食物一鍵添加（牛奶、雞蛋、水果等）
   - 智能類別自動識別

3. **惜食成就統計**
   - 已挽救食物件數
   - 節省金額估算
   - 減少碳排放量
   - 連續使用天數

4. **智能食譜推薦**
   - 根據即將過期的食材推薦食譜
   - 快速篩選：15分鐘、零浪費、健康
   - 常見剩菜料理建議

5. **本地數據儲存**
   - 使用 LocalStorage 保存所有數據
   - 無需帳號，開箱即用
   - 隱私優先設計

## 使用方式

直接在瀏覽器打開 `index.html` 即可使用。

### 快速開始
1. 輸入食物名稱（如「牛奶」）
2. 選擇類別和過期日期
3. 點擊新增
4. 隨時查看即將過期的食物
5. 食用後點擊 ✓ 標記，獲得成就！
