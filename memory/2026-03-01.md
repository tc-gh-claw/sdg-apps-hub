- Studio Ghibli 動畫中的海洋場景（《崖上的波兒》、《海潮之聲》）

3. 每日行動清單（拒絕一次性塑膠、海灘清理、永續海鮮選擇、分享知識）

5. 每日海洋冷知識（10 條輪播，涵蓋氧氣產生、塑膠污染、生態數據）
6. 行動記錄日誌（時間戳 + 影響描述）
7. 成就徽章系統（海洋新手、清理達人、海洋守護者等）
8. 慶祝動畫模態框（完成行動後的視覺反饋）

**技術特點：**
- 純 CSS 波浪動畫（SVG 背景 + CSS 位移）
- 動態氣泡背景（JS 生成隨機大小與速度的氣泡）

- 生物發光風格的強調色（青綠 #4ecdc4）
- 卡片滑動輪播（CSS scroll-snap）
- LocalStorage 數據持久化
