# GitHub Pages 設定指引

## 目標
令 SDG Apps 可以喺網上直接運行

## 設定步驟

### 步驟 1：開啟 GitHub Pages
1. 去 https://github.com/tc-gh-claw/sdg-apps-hub/settings/pages
2. 喺 "Source" 揀：
   - **Branch**: `main`  
   - **Folder**: `/docs`
3. 撳 "Save"

### 步驟 2：等幾分鐘
GitHub Pages 需要 1-5 分鐘先會生效

### 步驟 3：測試連結
設定好之後，啲 app 會喺呢啲網址運行：
